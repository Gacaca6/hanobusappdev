## [1.0.0] - 2026-04-27

### Added
- Real-time GPS bus tracking on an interactive map
- Arrival time estimation for every stop along a route
- Intelligent route planning between any two points
- Multi-language support: Kinyarwanda, French, and English
- Push notifications for delays and service disruptions
- Responsive web landing page
- Firebase backend integration (Firestore, Auth, Cloud Functions, FCM)
- Favorites for frequent stops and routes
- Offline-first Progressive Web App support
